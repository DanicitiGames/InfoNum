# Queee???

Sim, esse aplicativo mostra informações numéricas, dentre elas:
1. Raiz quadrada;
2. Número positivo ou negativo;
3. Número par ou ímpar;
4. Porcentagem (10%, 20% 30%...);
5. Como obter esse número apartir de adição, subtração e divisão.
